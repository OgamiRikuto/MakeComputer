# MakeComputer
Cで作成した簡単なコンピュータです。  
  
codelist.txtに使用可能な機械語一覧があります。  
機械語は2進数と10進数どちらでも使用できます。  
また、機械語に命名をして、命名した命令でプログラムすることもできます。  
なお、命名は実行ごとに行う必要があります。  

## 実行方法

### コンパイル方法
コンパイラを使用します。  
例
```bash
$ gcc -o computer computer.c
```

実行ファイル`computer`が生成されます。  

### 実行
macOSの場合は
```bash
./computer
```
Windowsの場合は
```bash
./computer.exe
```
で実行できます。

## 操作方法
実行すると,
```bash
命名(2進数):b, 命名(10進数):t, 命名一覧:s, 命名検索:f, 命名削除:d, 実行:r, コード説明:c, 終了:e
mode:
```
と表示されます。

- b : 2進数の機械語コードを入力し、命名します
- t : 10進数の機械語コードを入力し、命名します
- s : 命名の一覧を表示します
- f : 10進数の機械語機械語コードで命名した命令を検索します
- d : 命名した名前で名前を削除します
- r : プログラムに移ります(詳細を後述)
- c : codelist.txtの内容を表示します
- e : プログラムを終了します

## 実行について
実行モードではプログラムの前に2つ質問されます.  

1つ目が順次命令実行型か、プログラム型かです。  
順次命令実行型を選択すると、機械語コードを1つ書くたびに命令が実行され、レジスタの内容が表示されます。  
プログラム型を選択すると、機械語コードを連続で書くことができます。終了を意味する `0b11111111`(2進数)、`255`(10進数)を記述することでプログラムの終わりを示します。  
終了したら、一気に実行し、各命令ごとのレジスタの内容も一気に表示します。  

2つ目の質問は機械語コードの入力方法です。
2進数で入力するか、10進数で入力するか、命名した命令で入力するかです。  
2進数を選択した場合、1bitごとに空白を空けてください。
